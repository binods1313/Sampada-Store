## 📱 Responsive Behavior

### Desktop (> 1024px):
- Contact Methods: 2x2 grid
- Business Hours: 3 cards side-by-side
- Resources: 3 columns
- Hero quote: Left-aligned, 480px max-width

### Tablet (768px - 1024px):
- Contact Methods: 2x2 grid
- Business Hours: 3 cards (may wrap)
- Resources: 2 columns
- Hero quote: Same as desktop

### Mobile (< 768px):
- Contact Methods: 1 column
- Business Hours: 1 column
- Resources: 1 column
- Hero quote: Full width, 24px padding
- Hero image: object-position shifts to show face

## ✨ Interactive Elements

### Hover Effects (Desktop):
1. **Contact Cards**: Lift 4px, border glows gold
2. **Resource Cards**: Lift 4px, border glows gold, arrow appears
3. **FAQ Cards**: Question text turns gold
4. **CTA Button**: Background brightens, glows gold, scales 1.02

### Click/Tap Effects:
1. **FAQ Cards**: Expand/collapse with smooth animation
2. **Resource Cards**: Navigate to linked page
3. **CTA Button**: Navigate to contact/ticket page

### Animations:
1. **Snow Particles**: Continuous falling, gentle horizontal drift
2. **FAQ Chevron**: Rotates 180° on expand
3. **Card Hovers**: 0.3s ease transitions
4. **Button Hover**: 0.25s ease transition

## 🔍 What to Look For

### ✅ Correct Implementation:
- [ ] Model's face is visible in hero (not cropped)
- [ ] Snow particles are subtle (not overwhelming)
- [ ] Quote has gold left border (3px)
- [ ] All section titles have 60px gold underline
- [ ] FAQ cards have red left accent bar
- [ ] Cards have glass-morphism effect (slightly transparent)
- [ ] Hover effects work smoothly
- [ ] Sanskrit "स" watermark visible in CTA section
- [ ] All text is readable (good contrast)
- [ ] Mobile layout stacks properly

### ❌ Common Issues to Avoid:
- [ ] Hero overlay too dark (hides model)
- [ ] Snow effect too heavy (looks like blizzard)
- [ ] White or light backgrounds (breaks brand)
- [ ] FAQ as plain accordion (should be cards)
- [ ] Missing gold underline bars
- [ ] Missing red accent bars on FAQ
- [ ] Cards not lifting on hover
- [ ] Text too small on mobile
