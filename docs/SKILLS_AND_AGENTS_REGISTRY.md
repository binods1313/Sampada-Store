## ЁЯТб Usage Notes

**How to Use Skills:**
```
/emil-design-eng    # Invoke Emil Kowalski's design philosophy
/ui-ux-pro-max      # Get UI/UX design intelligence
/review             # Review code changes
/frontend-slides    # Create presentation slides
/loop 5m check build # Schedule recurring task
```

**How Agents Work:**
- I **automatically delegate** to the right agent when your task matches their specialty
- You can also **explicitly request** a specific agent by name
- Multiple agents can run **in parallel** for faster results

**Example:**
```
"Use the security-engineer to audit my authentication flow"
"Launch the china-market-localization-strategist for our China GTM plan"
"Run the gsd-roadmapper to plan this feature"
```
